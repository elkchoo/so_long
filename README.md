## Description

This project creates a simple game where you're a Roomba and you have to clean up the glitter and go to the exit

## Instructions

### Compilation

Compile the game with

```bash
make
```
or

```bash
make all
```

For the bonus version, compile with

```bash
make bonus
```

### Execution

For the base version:

```bash
./so_long (map file)
```

For bonus:

```bash
./so_long_bonus (map file)
```
